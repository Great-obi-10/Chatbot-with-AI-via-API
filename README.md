# AI Chatbot using FastAPI

This project is a simple AI chatbot that communicates with an AI model using an API.

Users type a message in the web interface and receive an AI-generated response.

## Installation

Clone the repository

git clone https://github.com/great-obi-10/ai-chatbot-project.git

cd ai-chatbot-project

---

Install dependencies

pip install -r requirements.txt

---

Create .env file

OPENAI_API_KEY=your_api_key_here

---

Run the server

uvicorn app:app --reload

---

Open browser

http://127.0.0.1:8000
